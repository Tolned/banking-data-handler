## 🚀 Архитектура и функциональные модули

### 🔍 1. Потоковая обработка и генераторы (`src/generators.py`)
Оптимизированный инструмент для работы с большими массивами финансовых данных без перегрузки оперативной памяти:
<ul>
  <li><code>filter_by_currency</code> — ленивый итератор, динамически фильтрующий транзакции по валюте (USD, RUB, EUR).</li>
  <li><code>transaction_descriptions</code> — генератор текстовых описаний для каждой банковской операции.</li>
  <li><code>card_number_generator</code> — бесконечный/диапазонный генератор маскированных номеров карт в формате <code>XXXX XXXX XXXX XXXX</code>.</li>
</ul>

### 📝 2. Декоратор автоматического контроля (`src/decorators.py`)
<ul>
  <li><code>@log</code> — многофункциональный декоратор для трекинга работы функций. Логирует время вызова, успешный результат или детали возникшей ошибки (Traceback).</li>
  <li><b>Гибкий вывод:</b> поддерживает запись как в терминал (stdout), так и в локальный текстовый файл при указании аргумента <code>filename</code>.</li>
</ul>

### 📊 3. Мультиканальный парсинг данных (`src/pandas_library.py`)
Обеспечивает интеграцию с аналитической библиотекой <b>Pandas</b> для чтения отчетов из различных банковских выгрузок:
<ul>
  <li><b>CSV-парсер:</b> автоматическая конвертация плоских таблиц в оптимизированные структуры Python.</li>
  <li><b>Excel-парсер:</b> полноценное считывание и валидация реестров операций из документов формата <code>.xlsx</code>.</li>
</ul>

### 🏷️ 4. Регулярные выражения и статистика
<ul>
  <li><b>Модуль регулярных выражений:</b> гибкий поиск и фильтрация транзакций по паттернам в описании (использование библиотеки <code>re</code>).</li>
  <li><b>Категоризация:</b> модуль подсчета операций определенного типа на базе высокопроизводительного класса <code>Counter</code>.</li>
</ul>

## 💻 Инструкция по развертыванию и запуску

### 1. Клонирование репозитория
```bash
git clone -b develop https://github.com
cd banking-data-handler
```

### 2. Инициализация виртуального окружения
```bash
# Установка изолированных зависимостей проекта через Poetry
poetry install
```

### 3. Запуск главной точки входа
```bash
# Запуск интерактивного пользовательского интерфейса приложения
poetry run python src/main.py
```
